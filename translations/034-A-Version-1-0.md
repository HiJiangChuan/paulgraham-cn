
# 随笔初版

## 2004年10月

每天收工前，我都会保存一份当天写作的备份。我想着，把一篇文章的定稿和最初的草稿对比一下，看看有多少原始文字得以留存，应该很有意思。正如 E. B. 怀特所说："好文章是改出来的。"上学的时候我并不明白这个道理。写作也好，数学也好，科学也好，呈现给学生的永远是最终成品，看不到那些走过的弯路。这让学生对事物的诞生过程产生了误解。

这种现象部分源于作者不愿让人看到自己的失误。但如果展示初稿能让人看清，要把一篇文章打磨成形需要多少轮改写，我愿意接受这种"暴露"。

以下是我能找到的[随笔的时代](https://hijiangchuan.com/paulgraham/032-The-Age-of-the-Essay)最早版本（大约是写作第二或第三天的状态）。最终留存下来的文字我用红色标注，后来被删除的文字用灰色标注。删减大致分几类：写错的内容、听起来像自吹自擂的内容、过激言论、离题的段落、笨拙的行文，以及多余的字眼。

开头部分删得最多。这不奇怪——找到节奏需要一点时间。开头的离题也更多，因为那时我还不确定要往哪里走。

整体删减量大约是平均水平。一篇文章的定稿，大概每一个字，我写草稿时就要写三到四个字。

（如果有人因为这里的某些观点感到不满，请记住：凡是没有出现在定稿里的内容，显然都是我选择不发表的东西，通常是因为我自己也不同意它。）

较小的零散删减：

写作有一个缺点，就是随笔可能包含错误（定理也一样）。优点是可以谈论更多的事情。

世界不是划分为"科学的"和"主观的"。科学是一种程度。有很多问题不适合用对照实验来回答，但仍然值得讨论。事实上，一些最有趣的问题就是这种类型。

对数学家来说，物理学家的工作显得相当推测性。

但如果精确性不那么严格，他们可以讨论更广泛的话题。

有时我会想在随笔开头加一个免责声明：如果你觉得无限定的陈述让你不舒服，就请在每个陈述前加上你习惯看到的"也许"。我的意思是一样的。

别指望人们会因为你向他们传递了这些宝贵的事实而感谢你——就像他们不会因为你给他们端上一盘清蒸蔬菜而感谢你一样。


这些是我最初的写作笔记，我通常在开始之前写不了这么多。我从没写到我打算涵盖的话题的一半，因为重心从散文风格转向了想法本身。
