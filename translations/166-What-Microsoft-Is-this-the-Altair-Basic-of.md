
# 这是哪个微软的 Altair Basic？

## 2015年2月

如果你想理解创业，最有价值的练习之一，就是去看那些最成功的公司，然后解释：为什么它们刚发布时看起来那么平庸？因为它们几乎无一例外地，最初看起来都很平庸。不只是小，而是平庸。不只是攀登大山的第一步，更像是走进沼泽的第一步。

为 Altair 写一个 Basic 解释器？那怎么可能长成一家巨型公司？让人们睡在陌生人公寓的气床上？一个让大学生互相盯梢的网站？一台用电视做显示器、供爱好者玩的小型单板电脑？当已经有大约10个搜索引擎、都在努力淡化搜索时，又一个新搜索引擎？这些想法不只是看起来小，而是看起来根本就是错的。是那种不仅可以被忽视、甚至会被嘲笑的想法。

通常连创始人自己都不知道为什么他们的想法有前景。他们被这些想法所吸引，凭的是直觉——因为他们[活在未来](https://hijiangchuan.com/paulgraham/151-How-to-Get-Startup-Ideas)，感觉到某种东西的缺失。但他们无法用语言准确说出，自己的这只丑小鸭将如何长成一只美丽的大天鹅。

大多数人听到一个听起来平庸的新初创想法，第一反应是嘲笑它。即便是很多本该更有见识的人，也会如此。

当我遇到一个想法听起来平庸的初创公司，我会问："这是哪个微软的 Altair Basic？"现在这成了一个谜题，而且解谜的责任在我。有时候我想不到答案，尤其是当那个想法是凭空捏造的。但令人惊讶的是，往往真的有答案。而且常常是创始人自己还没想到的那个答案。

有趣的是，有时候答案不止一个。几天前，我和一家初创公司聊，他们有可能长成三家截然不同的微软。这三个方向的规模可能相差几个数量级。但你永远无法预测一个"微软"会有多大，所以在这种情况下，我鼓励创始人追随当下让他们最兴奋的那条路。他们的直觉把他们带到了这里，何必现在停下来呢？
