
# 流行的问题

December 2019

我在许多不同的领域都看到了同样的模式：尽管很多人在这个领域努力工作，但只有一小部分可能性空间被探索过，因为他们都在研究相似的东西。

即使是最聪明、最富有想象力的人，在决定研究什么时也出人意料地保守。那些在其他方面绝不会追随潮流的人，却被卷入去研究流行的问题。

如果你想尝试研究不流行的问题，最好的寻找地方之一就是那些人们认为已经被充分探索过的领域：比如散文、Lisp、风险投资——你可能注意到这里有个模式。如果你能找到进入一个庞大但看似已经过时领域的新方法，你所发现的任何东西的价值都会因其巨大的表面积而被[成倍放大](https://hijiangchuan.com/paulgraham/179-General-and-Surprising)。

防止被卷入和其他人研究相同东西的最好保护，可能就是[真正热爱](https://hijiangchuan.com/paulgraham/180-The-Bus-Ticket-Theory-of-Genius)你所做的事情。这样即使你犯了和其他人一样的错误，认为它太边缘而不重要，你也会继续研究下去。
