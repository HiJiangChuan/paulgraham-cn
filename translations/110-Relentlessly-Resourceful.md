
# 坚韧且足智多谋

想要创业？来 [Y Combinator](http://ycombinator.com/apply.html) 获得资助吧。

2009年3月

几天前，我终于把成为一个优秀创业者的要素归纳为两个词：坚韧且足智多谋（relentlessly resourceful）。

在此之前，我最多只能把相反的品质归纳为一个词：无助（hapless）。大多数词典说 hapless 意味着不幸。但词典在这方面做得不太好。一支球队表现优于对手，却因为裁判的错误判决而输掉比赛，这可以称为不幸，但不是无助。无助意味着被动。无助就是被环境摆布，让世界随意支配你，而不是你去掌控世界。"不幸"是一个糟糕的同义词，因为它强调的是运气在坏结果中的作用。而无助更多地与坏结果的另一个组成部分重叠，即那些让坏结果显得理所应当的部分。[1]

不幸的是，hapless 没有反义词，这使得很难告诉创业者应该追求什么。"不要无助"并不是一个很有号召力的口号。

用隐喻来表达我们所寻找的品质并不难。最好的比喻可能是橄榄球跑卫。一个好的跑卫不仅要有决心，还要灵活。他们想要向前推进，但会随机应变地调整计划。

不幸的是，这只是一个隐喻，对美国以外的大多数人来说并不实用。"像跑卫一样"并不比"不要无助"更好。

但我终于想出了如何直接表达这种品质。我在为[投资人](https://hijiangchuan.com/paulgraham/109-How-to-Be-an-Angel-Investor)写一篇演讲稿，我必须解释在创业者身上应该寻找什么。与无助相反的人会是什么样子？他们会是坚韧且足智多谋的。不仅仅是坚韧。仅凭坚韧不足以让事情按你的意愿发展，除非在少数几个大多无趣的领域。在任何有趣的领域，困难都是新颖的。这意味着你不能简单地强行突破它们，因为你一开始并不知道它们有多难；你不知道你即将突破的是一块泡沫还是花岗岩。所以你必须足智多谋。你必须不断尝试新事物。

要坚韧且足智多谋。

这听起来不错，但这只是对如何在一般意义上取得成功的描述吗？我不这么认为。例如，这不是写作或绘画成功的秘诀。在那类工作中，秘诀更多的是保持积极的好奇心。足智多谋意味着障碍是外部的，这在创业中通常是这样。但在写作和绘画中，障碍大多是内部的；障碍是你自己的迟钝。[2]

可能还有其他领域，"坚韧且足智多谋"是成功的秘诀。但尽管其他领域可能与之相同，我认为这是我们能找到的对优秀创业者特质的最佳简短描述。我怀疑它能否更精确。

既然我们知道了我们在寻找什么，这就引出了其他问题。例如，这种品质可以教授吗？在尝试教授了四年之后，我想说，是的，令人惊讶的是，它经常可以被教授。不是对每个人，但对许多人来说可以。[3] 有些人天生就是被动的，但其他人有一种潜在的能力，可以变得坚韧且足智多谋，只需要被激发出来。

这对那些一直以来都处于某种权威之下的年轻人来说尤其如此。坚韧且足智多谋绝对不是在大公司或大多数学校取得成功的秘诀。我甚至不想去想大公司的成功秘诀是什么，但它肯定更长更复杂，涉及足智多谋、服从和建立联盟的某种组合。

识别这种品质也让我们更接近回答人们经常想知道的一个问题：可能有多少家创业公司。正如一些人似乎认为的那样，这个数字并没有经济上的上限。没有理由相信消费者能够吸收的新创造财富的数量有任何限制，就像可以证明的定理数量没有限制一样。所以创业公司数量的限制因素可能是潜在创始人的储备。有些人会成为好的创始人，而其他人则不会。现在我们可以说出是什么造就了一个好的创始人，我们就知道如何为这个储备的规模设定上限了。

这个测试对个人也很有用。如果你想知道自己是否是适合创业的那类人，问问自己是否坚韧且足智多谋。如果你想知道是否应该招募某人作为联合创始人，问问他们是否如此。

你甚至可以在战术上使用它。如果我在经营一家创业公司，这将是我贴在镜子上的短语。"做人们想要的东西"是目的地，但"坚韧且足智多谋"是你到达那里的方式。

注释

[1] 我认为词典错误的原因是这个词的含义已经发生了变化。今天从头开始编写词典的人不会说 hapless 意味着不幸。但在几百年前，他们可能会这么说。过去人们更受环境的摆布，因此我们用来描述好坏结果的许多词汇都源于关于运气的词汇。

当我住在意大利时，有一次我试图告诉某人我在做某事时没有取得多大成功，但我想不起意大利语中"成功"这个词。我花了一些时间试图描述我想表达的词。最后她说："啊！Fortuna！"

[2] 创业中也有一些方面，秘诀是保持积极的好奇心。有时你所做的几乎是纯粹的发现。不幸的是，这些时候只占整体的一小部分。另一方面，在研究中也是如此。

[3] 我几乎要说对大多数人来说都是如此，但我意识到（a）我不知道大多数人是什么样的，（b）我对人们改变能力的看法过于乐观。
