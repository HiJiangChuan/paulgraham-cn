
# 如何让匹兹堡成为创业中心

2016年4月

（这是我在匹兹堡一个名为 Opt412 的活动上的演讲。其中大部分内容也适用于其他城市。但不是全部，因为正如我在演讲中所说，匹兹堡相比大多数想成为创业中心的城市有一些重要优势。）

要让匹兹堡成为像硅谷一样的创业中心，需要什么条件？我对匹兹堡很了解，因为我在这里长大，在门罗维尔。我对硅谷也很了解，因为那是我现在住的地方。你能在这里建立起那样的创业生态系统吗？

当我同意来这里演讲时，我以为我无法给出一个很乐观的演讲。我以为我会谈论匹兹堡可以做什么来成为创业中心，非常多地使用虚拟语气。但现在我要谈的是匹兹堡能够做什么。

改变我想法的是我在《纽约时报》的美食版（没错，就是美食版）读到的一篇文章。标题是「[匹兹堡：年轻人带动的美食热潮](http://www.nytimes.com/2016/03/16/dining/pittsburgh-restaurants.html)」。对大多数人来说，这个标题可能都谈不上有趣，更别说和创业有什么关系了。但它读起来让我十分振奋。就算让我刻意去想，我也很难想出一个更有希望的标题。读完那篇文章，我更兴奋了。文章说：“25 至 29 岁的人如今占全体居民的 7.6%，而大约十年前是 7%。”哇，我想，匹兹堡可能会成为下一个波特兰——那个所有二十多岁的人都想去住的酷地方。

几天前我到这里时，我能感受到这种变化。我从1968年到1984年住在这里。当时我没有意识到，但在那整个时期，这座城市一直在自由落体。除了到处都在发生的郊区化迁移之外，钢铁和核能产业都在衰落。现在情况完全不同了。不仅仅是市中心看起来繁荣多了。这里有一种我小时候没有的活力。

我小时候，这是一个年轻人离开的地方。现在它是一个吸引年轻人的地方。

这与创业有什么关系？创业公司是由人组成的，而一个典型创业公司里的人的平均年龄正好在25至29岁这个区间。

我见证过一个城市拥有这些人的力量有多大。五年前，他们把硅谷的重心从半岛转移到了旧金山。谷歌和 Facebook 在半岛上，但下一代的大赢家都在旧金山。重心转移的原因是人才争夺战，尤其是程序员。大多数25至29岁的人想住在城市里，而不是无聊的郊区。所以不管创始人喜不喜欢，他们知道必须在城市里。我认识好几个创始人，他们本来更喜欢住在硅谷本部，但他们强迫自己搬到旧金山，因为他们知道否则会输掉人才争夺战。

所以成为吸引二十多岁年轻人的磁石是一件非常有前途的事情。很难想象一个地方能成为创业中心而不同时具备这一点。当我读到关于25至29岁人口比例增加的统计数据时，我的兴奋感与看到一家创业公司的图表开始从x轴向上爬升时的感觉完全一样。

全国25至29岁人口的比例是6.8%。这意味着你们领先0.8%。人口是30.6万，所以我们说的是大约2500人的盈余。那是一个小镇的人口，而这只是盈余。所以你们有了立足点。现在你们只需要扩大它。

虽然"年轻人带动的美食热潮"听起来可能很轻浮，但它绝非如此。餐厅和咖啡馆是一个城市个性的重要组成部分。想象一下走在巴黎的街道上。你走过的是什么？小餐馆和咖啡馆。想象一下开车穿过某个令人沮丧的随机远郊。你开过的是什么？星巴克、麦当劳和必胜客。正如格特鲁德·斯坦因所说，那里没有"那里"。你可能在任何地方。

这些独立的餐厅和咖啡馆不仅仅是在给人们提供食物。它们在让这里成为一个"这里"。

所以这是我把匹兹堡变成下一个硅谷的第一个具体建议：尽一切可能鼓励这种年轻人带动的美食热潮。城市能做什么？把开这些小餐馆和咖啡馆的人当作你的用户，去问他们想要什么。我能猜到他们可能想要的至少一件事：快速的许可证办理流程。旧金山在这方面给你们留下了很大的超越空间。

不过我知道餐厅不是原动力。正如《纽约时报》文章所说，原动力是便宜的住房。这是一个很大的优势。但"便宜的住房"这个说法有点误导人。有很多地方更便宜。匹兹堡的特别之处不在于它便宜，而在于它是一个你真的想住的便宜地方。

部分原因在于建筑本身。我很久以前就意识到了，那时我自己还是个穷困的二十多岁年轻人，最划算的地方是那些曾经富裕、后来变穷的地方。如果一个地方一直都很富裕，它很好但太贵了。如果一个地方一直都很穷，它很便宜但很糟糕。但如果一个地方曾经富裕后来变穷了，你可以便宜地找到宫殿。这就是吸引人们来这里的原因。一百年前匹兹堡富裕的时候，住在这里的人建造了大型坚固的建筑。不总是品味最好的，但绝对坚固。所以这里还有一条成为创业中心的建议：不要摧毁那些吸引人们来这里的建筑。当城市像匹兹堡现在这样正在复兴时，开发商会争相拆除旧建筑。不要让这种事发生。专注于历史保护。大型房地产开发项目不是吸引二十多岁年轻人来这里的东西。它们是新餐厅和咖啡馆的对立面；它们从城市中减去了个性。

经验证据表明，在历史保护方面你不可能太严格。城市在这方面越严格，似乎发展得越好。

但匹兹堡的吸引力不仅仅在于建筑本身。还在于它们所在的社区。像旧金山和纽约一样，匹兹堡幸运地是一座汽车时代之前的城市。它不会太分散。因为那些25至29岁的年轻人不喜欢开车。他们更喜欢步行、骑自行车或乘坐公共交通。如果你最近去过旧金山，你不可能不注意到大量的骑自行车的人。这不仅仅是二十多岁年轻人采用的一种时尚。在这方面，他们发现了一种更好的生活方式。胡子会消失，但自行车不会。不用开车就能到处走的城市就是更好，没有例外。所以我建议你们尽一切可能利用这一点。就像历史保护一样，似乎不可能做得太过分。

为什么不让匹兹堡成为全国对自行车和行人最友好的城市呢？看看你们能不能做到让旧金山相比之下显得落后。如果你们做到了，你们极不可能后悔。这座城市对你们想吸引的年轻人来说会像天堂一样。如果他们确实为了去别处工作而离开，他们会对离开这样一个地方感到遗憾。有什么坏处呢？你能想象一个标题"城市因变得对自行车太友好而毁掉"吗？这种事不会发生。

假设酷炫的老社区和酷炫的小餐馆让这里成为下一个波特兰。那够了吗？这会让你们处于比波特兰本身好得多的位置，因为匹兹堡有波特兰缺少的东西：一流的研究型大学。卡内基梅隆大学加上小咖啡馆意味着你们拥有的不仅仅是喝拿铁的文艺青年。它意味着你们有喝着拿铁讨论分布式系统的文艺青年。现在你们真的很接近旧金山了。

事实上，在一个方面你们比旧金山更有优势，因为卡内基梅隆大学在市中心，而斯坦福和伯克利在郊区。

卡内基梅隆大学能做什么来帮助匹兹堡成为创业中心？成为一所更好的研究型大学。卡内基梅隆大学是世界上最好的大学之一，但想象一下如果它是最好的，而且每个人都知道这一点，情况会怎样。有很多雄心勃勃的人必须去最好的地方，不管它在哪里。如果卡内基梅隆大学是那个地方，他们都会来这里。哈萨克斯坦会有孩子梦想有一天住在匹兹堡。

成为那种人才磁石是大学能为使其所在城市成为创业中心做出的最重要贡献。事实上，这几乎是它们能做出的唯一贡献。

但等等，大学不应该设立名称中带有"创新"和"创业"这些词的项目吗？不，它们不应该。这类事情几乎总是令人失望。它们追求的是错误的目标。获得创新的方法不是以创新为目标，而是以更具体的东西为目标，比如更好的电池或更好的3D打印。而学习创业的方法是去做，这是你[在学校里做不到的](https://hijiangchuan.com/paulgraham/157-Before-the-Startup)。

我知道一些管理人员可能会失望地听到，大学鼓励创业能做的最好的事情就是成为一所伟大的大学。这就像告诉想减肥的人，方法就是少吃。

但如果你想知道创业公司从哪里来，看看经验证据。看看最成功的创业公司的历史，你会发现它们是从几个创始人做一个一开始是有趣的副业项目中有机地成长起来的。大学很擅长把创始人聚集在一起，但除此之外，它们能做的最好的事情就是让开。例如，不要声称拥有学生和教职员工开发的"知识产权"的所有权，并且对延期入学和休假有宽松的规定。

事实上，大学鼓励创业能做的最有效的事情之一，是哈佛发明的一种精心设计的让开方式。哈佛过去在圣诞节后举行秋季学期的考试。在一月初，他们有一个叫做"阅读期"的东西，你应该在那时为考试学习。微软和 Facebook 有一个很少有人意识到的共同点：它们都是在阅读期开始的。这是产生那种变成创业公司的副业项目的完美情况。学生们都在校园里，但他们不必做任何事情，因为他们应该在为考试学习。

哈佛可能已经关闭了这扇窗户，因为几年前他们把考试移到了圣诞节前，并把阅读期从11天缩短到7天。但如果一所大学真的想帮助学生创业，按市值加权的经验证据表明，它们能做的最好的事情就是什么都不做。

匹兹堡的文化是它的另一个优势。看起来一个城市必须在社会观念上是自由开放的才能成为创业中心，原因很明显。一个城市必须容忍怪异才能成为创业公司的家园，因为创业公司是如此怪异。而且你不能选择只允许那些会变成大型创业公司的怪异形式，因为它们都是交织在一起的。你必须容忍所有的怪异。

这立即排除了[美国的大片地区](http://www.nytimes.com/2016/04/06/us/gay-rights-mississippi-north-carolina.html)。我乐观地认为它不会排除匹兹堡。我记得在这里长大的事情之一，虽然当时我没有意识到有什么不寻常的，就是人们相处得多么好。我仍然不确定为什么。也许一个原因是每个人都觉得自己是移民。当我还是门罗维尔的孩子时，人们不称自己为美国人。他们称自己为意大利人、塞尔维亚人或乌克兰人。想象一下一百年前这里是什么样子，当人们从二十个不同的国家涌入时。包容是唯一的选择。

我对匹兹堡文化的记忆是它既包容又务实。我也会这样描述硅谷的文化。这不是巧合，因为匹兹堡是它那个时代的硅谷。这是一个人们建造新事物的城市。虽然人们建造的东西已经改变了，但做这种工作所需要的精神是一样的。

所以，虽然涌入的喝拿铁的文艺青年在某些方面可能很烦人，但我会不遗余力地鼓励他们。更普遍地说，要容忍怪异，甚至达到古怪的加州人所做的程度。对匹兹堡来说，这是一个保守的选择：这是回归这座城市的根源。

不幸的是，我把最难的部分留到了最后。要成为创业中心，你还需要一样东西，而匹兹堡没有：投资者。硅谷有一个庞大的投资者社区，因为它有50年的时间来培育一个。纽约有一个庞大的投资者社区，因为它充满了非常喜欢钱并且能快速注意到赚钱新方法的人。但匹兹堡两者都没有。而吸引其他人来这里的便宜住房对投资者没有影响。

如果一个投资者社区在这里成长起来，它会像在硅谷一样发生：缓慢而有机地。所以我不会指望在短期内拥有一个庞大的投资者社区。但幸运的是，有三个趋势使得这比以前不那么必要了。一是创业公司的启动成本越来越低，所以你不需要像以前那样多的外部资金。第二是由于 Kickstarter 之类的东西，创业公司可以更快地获得收入。你可以从任何地方把东西放到 Kickstarter 上。第三是像 Y Combinator 这样的项目。来自世界任何地方的创业公司都可以去 YC 待3个月，获得资金，然后如果他们想的话可以回家。

我的建议是让匹兹堡成为创业的好地方，逐渐会有更多的创业公司留下来。其中一些会成功；他们的一些创始人会成为投资者；然后更多的创业公司会留下来。

这不是成为创业中心的快速路径。但它至少是一条路径，这是很少其他城市拥有的。而且你们也不必在此期间做出痛苦的牺牲。想想我建议你们应该做的事情。鼓励本地餐厅，保护旧建筑，利用密度优势，让卡内基梅隆大学成为最好的，提倡包容。这些都是让匹兹堡现在宜居的东西。我说的只是你们应该更多地做这些事情。

这是一个令人鼓舞的想法。如果匹兹堡成为创业中心的路径是更加成为它自己，那么它就有很大的成功机会。事实上，它可能是同等规模城市中机会最大的。这需要一些努力和很多时间，但如果有哪个城市能做到，那就是匹兹堡。
