
# 如何写出有用的文章

2020年2月

一篇文章应该是什么样的？大家会说，要有说服力。这是我们在学校时写作的文章标准。但我认为我们可以把目标定得更远大一些：一篇文章应该是**有用的**。

首先，有用意味着**正确**。但光正确还不够。把话说得含糊其辞，很容易做到"正确"。这是学术写作的通病。如果你对某个问题一无所知，只要说"这个问题很复杂，需要考虑很多因素，不能过于简单化"之类的话，你就永远不会出错。

这些话当然没错，但对读者毫无帮助。有用的写作，要在不出错的前提下，把话说得尽可能**有力**。

举个例子，说"派克峰在科罗拉多州中部附近"，比说它"在科罗拉多州某处"更有用。但如果我说它"正好在科罗拉多州正中心"，那就过头了，因为它其实偏东一点。

精确和正确像是两股对抗的力量。忽略其中一个，很容易满足另一个。空洞的学术写作的反面，是煽动家那种大胆却失实的修辞。有用的写作既要大胆，又要真实。

有用的写作还需要另外两点：它要告诉人们一些**重要**的事情，而且这些事情至少对一部分人来说是**新的**。

告诉人们新东西，并不总是让他们感到惊讶。有时候，你只是帮他们把潜意识里知道、却从未说出口的东西表达出来。事实上，这类洞见可能更有价值，因为它们往往更加根本。

把这些放在一起：**有用的写作，就是把真实且重要的新知，尽可能清晰地告诉读者。**

注意，这些都是程度问题。比如，你不可能指望一个观点对所有人都是新的。你的任何洞见，在全世界七十亿人里，大概至少已经有一个人想到过。但只要这个观点对很多读者来说是新的，就够了。

正确性、重要性、力度也是如此。实际上，这四个要素就像四个可以相乘的数，乘积就是"有用性"的得分。我知道这样说有点过于简化，但道理确实如此。


我相信我给出的这个公式——**重要性 + 新颖性 + 正确性 + 力度**——就是好文章的配方。但我得提醒你，这也是惹人生气的配方。

问题的根源在于新颖性。当你告诉人们一些他们不知道的事情时，他们并不总是感激你。有时候人们不知道某件事，恰恰是因为他们**不想**知道——通常是因为它与某个珍视的信念相矛盾。事实上，如果你在寻找新颖的观点，流行但错误的信念是很好的矿藏。每一个流行的错误信念周围，都会形成一片思想的[死区](https://hijiangchuan.com/paulgraham/181-Novelty-and-Heresy)，那里相对无人探索，因为那里的想法与流行信念相悖。

"力度"这个要素只会火上浇油。如果有什么比"珍视的假设被反驳"更让人恼火的，那就是被**斩钉截铁地**反驳。

再加上，如果你使用了莫里斯技巧，你的文章看起来会非常自信——对于不同意你的人来说，可能是**令人恼火地**自信。你显得自信是因为你确实自信：你作弊了，只发表了你有把握的东西。在试图反驳你的人看来，你似乎从不承认错误。其实你一直在承认错误，只不过是在发表**之前**，而不是之后。

如果你的文章又写得尽可能简单，那就更糟了。简洁是命令的口吻。如果你观察一个处于弱势的人传达坏消息，你会发现他们往往会用很多词来软化冲击。反过来，对人说话简短，多少就是对人无礼。

有时候，故意把话说得比你本意更弱是有效的——在你其实很确定的事情前面加个"也许"。但你会注意到，作家这样做的时候，通常会带着一种"你懂的"的眨眼暗示。

我不喜欢过多地这样做。整篇文章都用反讽语调会显得很刻意。我认为我们必须面对一个事实：**优雅和简慢，是同一件事的两个名字。**

你可能会想，如果你足够努力地确保文章正确，它就会无懈可击。某种程度上是这样——它会对**有效的**攻击免疫。但在实践中，这没什么安慰作用。

事实上，有用写作的"力度"要素会让你特别容易被**歪曲**。如果你把一个观点说得尽可能强而又不失真，那么任何人只要稍微夸大一点你说的话，它就变成假的了。

很多时候他们甚至不是故意的。如果你开始写文章，你会发现最令人惊讶的事情之一是：反对你的人很少反对你实际写的内容。他们会编造一些据说是你说的东西，然后反对那个。

如果有用的话，反击的方法是让对方引用你写的某个具体句子或段落——他们认为那是错的——并解释为什么错。我说"如果有用的话"，是因为他们从来不会照做。所以虽然这看起来可能把脱轨的讨论拉回正轨，但真相是它从一开始就没在正轨上。

你应该主动预防可能的误解吗？如果是**相当聪明且善意的人**可能产生的误解，那就应该。事实上，有时候先说一些稍有误导性的话再加上修正，比试图一步到位说对更好。这样可能更高效，也更能模拟这个观点被发现的过程。

但我不认为你应该在正文中主动预防**故意的**歪曲。文章是与诚实读者相遇的地方。你不想为了防范不诚实的人而在窗户上装铁栏杆，把自己的房子搞得像监狱。[1] 防范故意歪曲的地方是在尾注里。但别以为你能预测到所有的歪曲。当你说了一些人们不想听的话时，他们歪曲你的才智，就像他们为自己想做但知道不该做的事找借口时一样无穷无尽。我怀疑这根本就是同一种技能。


不过，还有多少文章等着我们去写呢？这个问题的答案，可能是我关于文章写作学到的最令人兴奋的东西：**几乎所有的文章都还没写出来。**

虽然[散文](https://hijiangchuan.com/paulgraham/032-The-Age-of-the-Essay)是一种古老的体裁，但它从未被精耕细作。在印刷时代，发表成本高昂，市场对散文的需求不足以支撑大量出版。如果你因为写其他东西（比如小说）而出名，你可以出散文集。或者你可以写书评，借题发挥表达自己的观点。但通往散文家的直接道路并不存在。这意味着散文写得很少，而写出来的那些往往局限于狭窄的主题。

现在，感谢互联网，路通了。任何人都可以在网上发表文章。也许你一开始默默无闻，但至少你可以**开始**。你不需要任何人的许可。

有时候，某个知识领域会沉寂多年，直到某种变化使它爆发。密码学对数论做了这件事。互联网正在对散文做这件事。

令人兴奋的不是还有很多东西可以写，而是还有很多东西可以**发现**。有一类观点最适合通过写文章来发现。如果大多数文章还没写出来，那么大多数这类观点也还没被发现。
