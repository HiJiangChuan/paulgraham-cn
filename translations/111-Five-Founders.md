
# 五位创始人

2009年4月

Inc 杂志最近问我,谁是我认为过去30年里最有趣的5位创业公司创始人。如何判断谁最有趣呢?最好的标准似乎是影响力:谁是对我影响最大的5个人?当我与我们投资的公司交流时,我会以谁为例?我发现自己在引用谁的话?

## 1. 史蒂夫·乔布斯

我猜史蒂夫不仅对我,而且对大多数人来说都是最有影响力的创始人。很多创业文化就是苹果文化。他是最早的年轻创始人。虽然"疯狂伟大"的概念在艺术领域已经存在,但在1980年代将其引入一家公司还是个新颖的想法。

更了不起的是,他30年来一直保持着吸引力。人们期待苹果新产品的方式,就像期待畅销小说家的新书一样。史蒂夫可能不是真的亲自设计它们,但如果他不是CEO,这些产品就不会诞生。

史蒂夫聪明而有干劲,但硅谷有很多这样的人。让他与众不同的是他的[设计品味](https://hijiangchuan.com/paulgraham/014-Taste-for-Makers)。在他之前,大多数公司把设计当作无关紧要的附加品。苹果的竞争对手现在知道得更清楚了。

## 2. TJ·罗杰斯

TJ·罗杰斯不像史蒂夫·乔布斯那么有名,但他可能是硅谷CEO中最好的作家。我从他那里学到的关于创业思维方式的东西,可能比任何人都多。与其说是从他写的具体内容中学到的,不如说是通过重构产生这些内容的思维:残酷的坦诚;积极清除过时的想法;但又是由实用主义而非意识形态驱动的。

我读到他的第一篇文章时非常震撼,以至于我还记得当时在哪里。那篇文章是[《高科技创新:自由市场还是政府补贴?》](http://www.cypress.com/?rID=34993),我当时在哈佛广场地铁站的楼下。感觉就像有人在我脑子里打开了一个电灯开关。

## 3. 拉里和谢尔盖

很抱歉把拉里和谢尔盖当作一个人来对待。我一直觉得这对他们不公平。但谷歌确实看起来像是一次合作的产物。

在谷歌之前,硅谷的公司已经知道拥有最好的黑客很重要。至少他们是这么声称的。但谷歌把这个想法推进得比任何人都远。他们的假设似乎是,至少在初期阶段,你只需要优秀的黑客:如果你雇佣所有最聪明的人,让他们去解决一个可以衡量成功的问题,你就赢了。所有其他的东西——包括商学院认为商业所包含的所有东西——你可以在过程中慢慢摸索。结果不会完美,但会是最优的。如果这是他们的假设,现在已经通过实验得到了验证。

## 4. 保罗·布克海特

很少有人知道这一点,但有一个人,保罗·布克海特,负责了谷歌做过的三件最好的事情。他是 GMail 的原始作者,这是谷歌除搜索之外最令人印象深刻的产品。他还写了 AdSense 的第一个原型,并且是谷歌信条"不作恶"的作者。

PB 在一次演讲中提出了一个观点,我现在会向我们投资的每家创业公司提到:最初,让少数用户真正爱你,比让大量用户有点喜欢你要好。如果我只能告诉创业公司[十句话](https://hijiangchuan.com/paulgraham/105-Startups-in-13-Sentences),这会是其中之一。

现在他是一家叫 Friendfeed 的创业公司的联合创始人。它才成立一年,但硅谷的每个人都在关注他们。一个负责谷歌三个最大创意的人,将来会想出更多。

## 5. 萨姆·奥特曼

有人告诉我不应该在这个名单中提到 YC 投资公司的创始人。但萨姆·奥特曼不会被这种脆弱的规则阻止。如果他想上这个名单,他就会上。

老实说,萨姆和史蒂夫·乔布斯一样,是我在建议创业公司时最常提到的创始人。在设计问题上,我会问"史蒂夫会怎么做?"但在战略或野心问题上,我会问"Sama 会怎么做?"

我从见到 Sama 那里学到的是,天选之人的理论适用于创业公司。它的适用范围比大多数人想象的要小得多:创业投资并不是像赛马那样试图挑选赢家。但确实有少数人意志力如此强大,他们会得到任何他们想要的东西。
