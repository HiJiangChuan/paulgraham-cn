
# 有孩子之后

December 2019

在我有孩子之前，我很怕有孩子。那时我对孩子的态度，就像年轻的奥古斯丁对“过有德行的生活”的态度：一想到自己可能永远不会有孩子，我会难过；但要我现在就要吗？不要。

如果我有了孩子，我就成了父母。而父母——从我小时候起我就觉得——一点也不酷：无聊、负责任、没什么乐趣。孩子会这么想并不奇怪；但老实说，成年后的观察也没怎么改变我。每当我注意到带孩子的父母时，孩子都像小魔王，父母则像可怜又被折腾得焦头烂额的人，即便最后把场面镇住了也一样。

当别人有了宝宝时，我会热情地祝贺，因为好像就该这么做；但我心里一点也不这么想。我想的是：“幸好不是我。”

现在别人有了宝宝时，我也会热情地祝贺，而且是真心的。尤其是第一胎。我总觉得他们刚收到了世界上最好的礼物。

当然，变的是我：我自己有了孩子。曾经让我害怕的事，结果竟然美好得出奇。

一部分原因——我不否认——是我们第一个孩子出生时，我体内几乎瞬间发生了剧烈的化学变化。就像有人啪地拨了一下开关。我突然不只想保护自己的孩子，也对所有孩子都生出保护欲。那天我开车载着妻子和刚出生的儿子从医院回家，驶近一个挤满行人的斑马线时，我发现自己在想：“我得特别小心这些人。他们每一个都是某个人的孩子！”

所以在某种程度上，当我说有孩子很棒时，你不能完全相信我。在某种程度上，我就像某个宗教邪教的信徒，告诉你“你也加入就会很快乐”——但那只是因为加入会改变你的头脑，让你乐于做一名邪教成员。

但也不完全是这样。关于有孩子，有些事在我有他们之前显然理解错了。

比如，我对父母和孩子的观察有巨大的选择偏差。有些父母可能已经注意到我写的是“每当我注意到带孩子的父母时”。我当然只会在事情出问题时注意到孩子：他们吵闹时我才会注意到。那我是在什么场景下注意到他们？通常我根本不去孩子多的地方，所以真正遇到他们，多半是在飞机这种不得不共享的狭窄场景里。这当然毫无代表性。带着幼儿坐飞机，是极少有父母会享受的事。

而我没注意到的——因为这些时候通常安静得多——是父母和孩子之间那些美好时刻。人们不怎么谈这些——那种魔力很难用语言说清，而且别的父母也都懂——但有孩子的一大好处是：有很多时候，你会觉得没有别的地方比这里更想待，也没有别的事比此刻更想做。你不必在做什么特别的事情：可能只是一起去某个地方，或者哄他们睡觉，或者在公园推他们荡秋千。但你不会用任何东西去交换这些时刻。我们很少把孩子和“宁静”联系在一起，可你当下感到的恰恰是宁静。你无需再往别处看——现在这里就够了。

在有孩子之前，我也有过这种宁静的时刻，但更少。有了孩子，这种时刻一天可能出现好几次。

我关于孩子的另一个“数据来源”是我自己的童年，这同样具有误导性。我小时候挺糟糕，总因为这事那事惹麻烦。所以在我看来，做父母基本等同于执法。我没意识到其中也有好时光。

我记得大概我30岁时，母亲曾说她很享受有我和我妹妹。天哪，我当时想，这女人是个圣人：我们折腾她那么多，她不但忍了，还说享受？现在我明白，她只是在讲实话。

她说她喜欢有我们，其中一个原因是我们是很有意思的聊天对象。等我自己有了孩子，这点让我吃惊。你不只是爱他们——他们也会成为你的朋友。他们真的很有趣。虽然我承认小孩子对重复的热爱到了灾难级别（任何值得做一次的事都值得做五十次），但跟他们玩往往确实很好玩。这个也让我意外。两岁时跟两岁孩子玩很好玩，六岁时肯定不好玩；为什么后来又会变得好玩？但事实就是如此。

当然，也有纯粹的苦役时刻，甚至更糟。有孩子是一种强烈到你没经历过就很难想象的体验。但它并不是我在有孩子之前暗暗以为的那样，只是基因在“自救”——为了延续自身而本能地往救生艇上挤。

不过，我的一些担心确实没错。有孩子肯定会让你没那么高产。我知道有孩子会让某些人“振作起来”，但如果你本来就已经很自律，你只会发现可用时间更少。尤其是，你得按时间表工作，孩子的时间表。我不确定这是因为孩子天生如此，还是因为要把他们的生活和成年人的生活拼到一起只能如此；但一旦有了孩子，你往往就得跟着他们的节奏走。

你当然还是会有一块块可用来工作的时间。但你不能再像我生孩子前那样，让工作随意地渗透到生活的每个角落。你得每天在固定时间工作，不管灵感来不来；而且有时候，即使灵感正涌着，你也得停下来。

我已经适应了这种工作方式。工作和爱情一样，总会找到出路：如果它只能在某些时间发生，它就会在那些时间发生。所以虽然我做不到生孩子前那么多，但也做得够多。

我很不想这么说，因为雄心一直是我身份的一部分，但有孩子可能会让人没那么雄心勃勃。把这句话写出来都疼，我会扭来扭去想躲开它。但如果不是戳中了点什么，我为什么要躲？事实是，一旦有了孩子，你多半会更在乎他们胜过自己。而注意力是零和的：同一时刻，你脑子里只能有一个想法是[你头脑中的首要想法](https://hijiangchuan.com/paulgraham/127-The-Top-Idea-in-Your-Mind)。有了孩子之后，它往往是孩子——这意味着它就更少是你正在做的某个项目。

我也有一些小办法，尽量贴着这股风的边航行。比如我写文章时，会想：我希望孩子知道什么？这会逼着我把事情讲对。写 [Bel](https://hijiangchuan.com/paulgraham/EXTRA011-Bel) 时，我对孩子们说，等我写完就带他们去非洲。你跟小孩子说这种话，他们会当成承诺。这就意味着我必须写完，不然就等于夺走了他们的非洲之旅。也许如果我真的很走运，这些小技巧能让我“净领先”。但那股风确实在，毫无疑问。

话又说回来，如果你的雄心一有孩子就活不下去，那你的雄心也太脆弱了吧？你到底有多少余量？

尽管有孩子可能在扭曲我眼下的判断，但它没有抹掉我的记忆。我清楚记得有孩子之前的生活。清楚到会很想念某些东西，比如随时收拾行李飞去另一个国家的能力。那太爽了。可我为什么从来没这么做过？

看到我刚才做了什么吗？事实是：有孩子之前我拥有的大部分自由，我从没用过。我用孤独为它付了价，却没享受它。

有孩子之前我也有很多快乐的时光。但如果我把快乐的瞬间——不是“可能的快乐”，而是实际发生的快乐——数一数，有孩子之后反而更多。现在我几乎随手就能取用，几乎每个哄睡时间都算。

做父母的体验差异很大，我知道自己很幸运。但我想，我在有孩子之前的担心一定很普遍；而从别的父母看孩子时的表情来看，孩子带来的快乐大概也同样普遍。
